## 📋 下一步操作

### 1️⃣ 打开 GitHub Secrets 页面

点击访问：**https://github.com/ashun520/CHAT/settings/secrets/actions**

### 2️⃣ 添加 3 个 Secrets

#### Secret 1: IOS_P12_BASE64
1. 点击 "New repository secret"
2. Name: `IOS_P12_BASE64`
3. Value: 打开 `ios_p12_base64.txt`，**全选复制所有内容**，粘贴到这里
4. 点击 "Add secret"

#### Secret 2: IOS_P12_PASSWORD
1. 点击 "New repository secret"
2. Name: `IOS_P12_PASSWORD`
3. Value: `1`
4. 点击 "Add secret"

#### Secret 3: IOS_MOBILE_PROVISION_BASE64
1. 点击 "New repository secret"
2. Name: `IOS_MOBILE_PROVISION_BASE64`
3. Value: 打开 `ios_mobileprovision_base64.txt`，**全选复制所有内容**，粘贴到这里
4. 点击 "Add secret"

### 3️⃣ 触发构建

添加完所有 Secrets 后：

1. 访问：**https://github.com/ashun520/CHAT/actions**
2. 点击左侧 "Build iOS IPA"
3. 点击 "Run workflow" 按钮
4. 选择分支 "main"
5. 点击 "Run workflow"

### 4️⃣ 下载 IPA

构建成功后（大约 5-10 分钟）：

1. 访问：**https://github.com/ashun520/CHAT/releases**
2. 下载最新的 IPA 文件
3. 使用爱思助手、AltStore 或 Xcode 安装到 iPhone

## 📱 安装方法

### 方法 1：爱思助手（最简单）
1. 下载安装爱思助手：https://www.i4.cn/
2. 连接 iPhone 到电脑
3. 打开爱思助手
4. 点击 "应用游戏" → "导入" → 选择 IPA 文件
5. 点击 "安装到设备"

### 方法 2：AltStore
1. 下载 AltStore：https://altstore.io/
2. 安装 AltStore 到 iPhone
3. 打开 AltStore
4. 点击右上角 "+"
5. 选择 IPA 文件

### 方法 3：Xcode
1. 打开 Xcode
2. Window → Devices and Simulators
3. 选择您的设备
4. 将 IPA 文件拖到 "Installed Apps" 区域

## ⚠️ 注意事项

- ✅ 确保证书密码是 `1`
- ✅ Provisioning Profile 必须包含您的设备 UDID
- ✅ 如果是 Ad Hoc 证书，只能安装到注册的设备
- ✅ 如果是企业证书，可以安装到任何设备

## 🔍 如何查看设备 UDID

1. 连接 iPhone 到电脑
2. 打开 iTunes 或 Finder
3. 点击设备图标
4. 点击 "序列号" 区域（会切换显示 UDID）
5. 右键复制 UDID

## 📞 有问题？

如果构建失败，请查看：
- GitHub Actions 日志：https://github.com/ashun520/CHAT/actions
- 检查证书是否过期
- 检查 Provisioning Profile 是否有效
