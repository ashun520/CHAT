====================================
  GitHub Secrets 配置指南
====================================

请访问：https://github.com/ashun520/CHAT/settings/secrets/actions

添加以下 3 个 Secrets：

1. IOS_P12_BASE64
   值：复制 ios_p12_base64.txt 的全部内容

2. IOS_P12_PASSWORD
   值：1

3. IOS_MOBILE_PROVISION_BASE64
   值：复制 ios_mobileprovision_base64.txt 的全部内容
